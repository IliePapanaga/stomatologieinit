5.1 Unblacklist a Dental Office

A professional can make unblacklist the dental office.

If a user would like to observe a list of blacklisted location, then he/she need to navigate to Banned Offices View.

  1. Navigate to Banned Offices at Assignment.
  2. Select a blacklisted dental office.
  3. Click on UNBLACKLIST button.
  4. Click on OK button.
    Remember: If a dental office (location) is in Blacklist, no postings are available for a professional.
