8. No Shows and Rejections Policy

According to Terms and Conditions Mayday Dental Staffing supports next notation:

No Show

It means that a candidate has not arrived into the Dental Office as per accepted job offering and a Doctor has marked this status.

Attention: If a candidate has two “No Shows”, a candidate is able to login into the system but able to see any previously accepted postings. A candidate does not receive any new posting notification. A candidate should communicate with Mayday Dental Staffing, if one wants use Mayday Dental Staffing system in future.

Rejection

A candidate is able to reject:

  • Previously accepted assignments. In this case, the application applies the penalty.

Attention: If a candidate has five Rejection, a candidate is able to login into the system but able to see any previously accepted postings. A candidate does not receive any new posting notification. A candidate should communicate with Mayday Dental Staffing, if one wants use Mayday Dental Staffing system in future.
