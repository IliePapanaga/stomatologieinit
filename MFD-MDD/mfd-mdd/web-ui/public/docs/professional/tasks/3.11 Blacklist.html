3.11 Blacklist

A user is able to add a dental office into a blacklist.
Note: If a dental office is in blacklist, no posting from the location are displayed for a professional.
  1. Navigate to Temporary Job at Assignment.
  2. Select Assignment.
  3. Click BLACKLIST button.
  • A dental office is in blacklist.
    Note: To make unblacklist a dental office, please navigate to Temporary Job at Banned Offices.
