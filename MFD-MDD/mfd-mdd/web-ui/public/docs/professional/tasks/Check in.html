3.8 Check In

A professional needs check in in a doctor’s location after arriving.

Note: A doctor is also able to make check in of a professional.
Note: If a doctor or a professional do not make check in and any other actions are done, the application make Checked In Status automatically within one hour after job starts.

  1. Navigate to Temporary Job at Assignment.
  2. Select an Assignment in a Need check in Status.
  3. Observe CHECK IN dialog.
  4. Click on OK button.
