4.3 Apply Permanent Posting

A professional is able to apply for a posting in “Active” status

  1. Navigate to Permanent Job at Assignment.
  2. Select an Assignment in a Active Status.
  3. Click on APPLY button.
  4. Observe POSTING dialog.
  5. Click VIEW button to review work schedule.
  6. Click CLOSE button.
  7. Click APPLY button.
  8. Observe that a posting has You have applied status.
