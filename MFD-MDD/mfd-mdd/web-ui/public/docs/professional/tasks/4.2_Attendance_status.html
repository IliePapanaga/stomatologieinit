4.2 Permanent Posting Statuses

.

Next Permanent Posting Statuses are supported

Table 1.
Status Description
Active An assignment is placed by a Dental Practice and a start date in future. An assignment is available for all eligible candidates for the specified time and date.
You have applied A candidate has applied to the assignment, however the Dental Practice has not approved yet an applicant.

A candidate can withdraw his application without any penalties.

The assignment is still available for all eligible candidates.
You are invited The Dental Practice has invited a candidate for interview (personal or working).The Candidate can accept or cancel interview date and time. The assignment is still available for all eligible candidates.
Interview is scheduled The Candidate accepted interview date and time.
You have an offer The Dental Practice has booked a candidate for a job. The assignment is still available for all eligible candidates.
Accepted The Candidate has accepted a job offer.

The Candidate is not able to accept more than one invitation to a job for the same date and time.

The assignment is not available for the rest of eligible candidates.
