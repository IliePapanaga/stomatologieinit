3.7 Accept Posting Dialog

After confirmation of a candidate by a dental office, a user is able to accept posting.

Attention: Please note that if a dental office asks you to work extra days not published in the assignment, you are obligated to notify Mayday Dental Staffing immediately of such extensions on days whether temporary or permanently by calling (925) 300-5734 or by email: contact@maydaydentalstaffing.com.
  1. Navigate to Temporary Job at Assignment.
  2. Select an Assignment in a You are confirmed Status.
  3. Click on APPLY button.
  4. Observe Temporary Assignements View.
  • A professional has accepted the job.
  • A posting has status Accepted.
