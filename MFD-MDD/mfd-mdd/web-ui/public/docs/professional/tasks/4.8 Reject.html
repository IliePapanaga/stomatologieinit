4.8 Reject

A doctor is able to hire the same candidate.

  1. Navigate to Permanent Job at Assignment.
  2. Select an Assignment in a You are confirmed or Accepted Status.
  3. Click on REJECT button.
  4. Note: Please be informed that after five rejections, penalties by Mayday Dental Staffing are applied and you can be excluded from the application users.
    Click on OK button.
