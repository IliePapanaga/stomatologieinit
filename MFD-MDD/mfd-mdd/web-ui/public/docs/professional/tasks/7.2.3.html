7.2.3 Edit Questionnaire

Each category has appropriate Questionnaire and a candidate need to add information about it.

  1. Navigate Specilaty Tab at the Navigation Panel..
  2. Select a specialty.
  3. Click EDIT QUESTIONNAIRE button.
    For example, for Assitants.
  4. Populate/select/check required fields/sections.
  5. Click OK button.
    Note: The information provided in EDIT QUESTIONNAIRE is available for dental practices and help them to make select a candidate.
