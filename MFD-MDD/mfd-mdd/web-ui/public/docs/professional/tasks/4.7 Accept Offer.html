4.7 Accept Offer

If a candidate is ready to accept the offer, one should select ACCEPT OFFER button.

Attention: Please note that if a dental office asks you to work extra days not published in the assignment, you are obligated to notify Mayday Dental Staffing immediately of such extensions on days whether temporary or permanently by calling (925) 300-5734 or by email: contact@maydaydentalstaffing.com.
  1. Navigate to Permanent Job at Assignment.
  2. Select an Assignment in You have an offer Status.
  3. Click ACCEPT OFFER button.
  4. Observe ACCEPT OFFER dialog.
  5. Click OK button.
Verify that a posting is in Accepted Status.
