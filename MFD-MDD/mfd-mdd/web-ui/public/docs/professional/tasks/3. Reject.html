3.9 Reject

A professional is able to reject the previously accepted job.

  1. Navigate to Temporary Job at Assignment.
  2. Select an Assignment in a You are confirmed or Accepted Status.
  3. Click on REJECT button.
  4. Note: Please be informed that after five rejections, penalties by MayDayDental Staffing are applied and a professional can be excluded from the application users or professional will not be able to see new postings.
    Click on OK button.
