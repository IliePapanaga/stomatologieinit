4.6 Cancel Interview

In some cases, a doctor would like to alert a candidate.

A professional is able to cancel the previosuly accepted interview.

  1. Navigate to Permanent Job at Assignment.
  2. Select an Assignment in Interview is scheduled Status.
  3. Click CANCEL INTERVIEW button.
  4. Observe CANCEL INTERVIEW dialog.
  5. Click OK button.
Verify that a posting is in You have applied Status.
