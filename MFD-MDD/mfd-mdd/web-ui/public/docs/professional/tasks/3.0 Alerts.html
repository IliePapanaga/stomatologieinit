3.13 Alerts

Alert is a type of notification to inform a professional that a dental office demonstrates some concerns about a professional arrival in time.

Note: A professional is able to observe alerts for an assignment, if:
  1. The Assignment is for the current day.
  2. A Doctor sends an alert to a professional.
  3. Status of Job is “Need check in”.
    Note: A doctor sends a notification to the candidate by sms/email and a professional is able to view the sms/email notification.
    Note: A professional cannot view the notifications in the application, it’s possible view/read the alerts in a phone (sms) or in emails.

    Attention: A professional cannot first send an alert until the doctor sends a request.
  1. Navigate to Temporary Job at Assignment.
  2. Select an Assignment with Need check in status and red bell icon
  3. Select an approriate answer.
    Note: A professional is able to respond a doctor using predefined templates, like:
    • Arriving in a minute;
    • Couple of minutes late;
    • Stack in traffic, will be in an hour;
    • So sorry, cannot come today.
  4. The notification is sent to a dental office.
