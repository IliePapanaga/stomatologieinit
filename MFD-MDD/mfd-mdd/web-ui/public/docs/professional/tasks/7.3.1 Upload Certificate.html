7.3.1 Upload Certificate

Each specialty has a predefined set of the required documentation. Some documents are not mandatory.
Note: If a certificate is not approved, a candidate is not able to participate in searching of assignments.

Attention: If any of the mandatory certificate date is expired, a candidate is not able to receive any new assignments.
  1. Navigate to Certificates at the Settings section at the left navigation panel.
  2. Select a required certificate in Pending Status.
  3. Click UPLOAD CERTIFICATE button.
  4. Observe UPLOAD CERTIFICATE Dialog.
    For examle, for XRAY Certificate:

  5. Click UPLOAD button to upload a required certificate.
    Note: Some Certificate Dialog requires Expiration date.
    Note: Please note that Status and Comments fields are updated after Mayday Dental Staffing review the provided documentation.
  6. Click CLOSE button.
  7. Observe that a record has Requires review status.
    See a list of supported statuses in 7.3.3 Certificate Statuses
