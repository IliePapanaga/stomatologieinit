3.3 Temporary Assignments Statuses

Next Statuses are supported in the application

Table 1. Temporary Posting Statuses
Status Description
Active An assignment is placed by the dental offices. An assignment is available for all eligible candidates for this time and date. A Candidate is able to apply or move the office to the blacklist.
You have applied A candidate applied to the assignment, however the Dental Practice has not approved yet an applicant. The candidate can withdraw his/her application without any penalties. Posting is still available for all eligible candidates.
You are confirmed

The Dental Practice has approved the candidate application. The Candidate can accept or reject the job offer. However, after 5 rejections, penalties take place.

The assignment is still available for all eligible candidates.
Accepted The candidate has accepted posting. The assignment is not available for the rest of eligible candidates.
Need check in

This is a gentle reminder for a candidate to check in the medical office. A candidate is able to make Check in manually one hour before and one hour after job starts.
Checked in A Candidate has arrived at the Dental Office and he/she checked in the office.
Note: If a professional accepted some job invitation earlier, it means one can not apply/accept to another posting for the same date and time or with overlapping periods. The previously accepted job offer needs to be rejected and after that, “Accept” button is enabled for a user. However, a rejection action after the assignment was accepted leads to accumulating penalties. After 5 rejections, a user will not be able to search for new assignments. One needs to contact Mayday Dental Staffing to be unblocked.
