7.4 Profile

A candidate need to provide “Skills Summary”, “Education” level, about “Highest Degree Earned” and information about “Languages” that you can use in your professional activity.

Note: Providing information about work experience and professional reference, usually, increase candidate’s ability to be selected by a dental office for an assignment.

  1. Navigate to Profile at the Settings section at the left navigation panel.
  2. Observe Profile dialog.
  3. Provide Skills Summary section.
  4. Select Education level.
  5. Select Highest Degree Earned.
  6. Select Languages.
    Note: It is possible to select several options for Languages.
  7. Provide Work experience.
  8. Provide Professionall reference.
  9. Click SAVE.
    Note: After completion all forms, Mayday Dental Staffing is notified about it and a review process is started.
