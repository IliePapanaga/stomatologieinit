3. Posting Management

After a user is logged into, a user observe Postings in Progress for temporary job. There are two available Tabs at the navigation panel: Temporary Job and Permanent Job.

Note: Before publish any temporary or permanent posting, a user should add, at least, one Location and one Payment Method.

Temporary posting is such kind of posting where you need a professional occasionally.

Permanent posting is such kind of posting where you need a professional for a long time, usually, more than one year.

Important:
Note: Please be aware that Temporary postings are charged on daily basis and Permanent postings are charged once.
