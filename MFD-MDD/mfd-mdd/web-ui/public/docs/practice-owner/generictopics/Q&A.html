Questions and Answers

Question: I have registered in the application but cannot place a posting. What is wrong?

Answer: Please check at the “Settings”, if you have added at least one location and one payment method.

Question: As a dental practice owner, I would not like that my posting are seen by some professionals. What can I do?

Answer: A doctor is also able to banned list a candidate if one has already worked for the office. Please go to Past hiring history, select the candidate and banned list one. After that the professional is not able to receive any notifications/postings about published vacancy by the dental practice.

Question: As a dental practice owner, I would like to interview the potential candidates for permanent posting before hiring. Can I do that?

Answer: A doctor is able to conduct working or personal interview with a candidate before hiring. Mayday Dental Staffing charges only working interview and monitors how many interviews are scheduled. A doctor cannot schedule an interview later that a start date of a job. In this case, a dental practice owner should “Edit” a posting to change a start date or “Delete” posting at all, and create a new one.
