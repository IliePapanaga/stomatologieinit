Troubleshooting the error messages

Environment cannot be deleted. Clean all referenced service(s)

You need to make sure the service templates associated with an environment are deleted. For details, see .4.4 Alerts

Cannot stop service. It does not contain Shutdown orchestration operation

You cannot stop a service if the service was not pre-configured to be stopped. For details, see 4.3 Rejection of a Candidate.
