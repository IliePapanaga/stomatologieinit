Adding a hardware platform

You can specify the resources that will be allocated directly to a VM when provisioning a service (i.e. the cloud provider's hardware resources a VM can access).

  1. Click Infrastructure Operations.
  2. Click the Deployment view, then the Hardware Platforms tab, and Add Hardware Platform.
  3. Enter the following:
    • Name - a platform's name.
    • Class - the description of a hardware platform's level, for example, ERP, medium, small business.
    • CPU, RAM, HDD - the resources to be allocated to a VM, including the number of core CPUs.
  4. Click OK.
The hardware platform you have set up is now available from Design Studio > Service Templates > Add Template > Deployment Profiles > Add Deployment Profile > Default Deployment Group > Hardware Platform.
