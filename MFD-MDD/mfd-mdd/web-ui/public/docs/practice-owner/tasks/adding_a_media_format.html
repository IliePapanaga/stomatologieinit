3.1.5 Delete Posting

You can specify the media format to be used by a service template, for example, .doc, .exe, .rar, etc. A media format is used exclusively for information-mapping and visualization purposes (so that you could see the list of available formats at a glance): it does not affect the functionality.

  1. Click Design Studio.
  2. Click the Classification view, the Media Formats tab, and Add Media Format.
  3. Enter the following:
    • Name - the format, e.g. .exe.
    • Mime Type - PLACEHOLDER.
  4. Click OK.
