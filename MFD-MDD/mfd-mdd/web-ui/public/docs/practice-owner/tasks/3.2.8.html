3.2.8 Book a Candidate

Remember: Please be informed that for permanent position you be charge according to the rules specified in 6.2 Payments for Permanent Positions
  1. Select a required Posting.
  2. Click View Applicants.
    Note: If no professional applied to a posting, View Applicants is disabled.
  3. Select a required Professional.
    Remember: A Doctor is able to HIRE a professional without conducting an interview.
  4. Click HIRE button at the Booking Column.
  5. Confirm your decision. Click OK button.
  6. Observe that Booking Status is Offer sent
  7. Click CLOSE button.
    Note: Until a professional does not confirm that one acceptes the offer, a Status for the posting is still Active.


  8. Observe a posting Status
    Note: When a professional accepts the offer, a Status is changed to Filled.
