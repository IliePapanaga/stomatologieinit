7.2.3 Address for Location is not found

In some cases, after providing a ZIP code, the address cannot be found and auto population of the address information does not work. In such cases, please use FIND MY LOCATION button.

  1. Click FIND MY LOCATION button.
  2. Observe that LOCATION dialog appears.
  3. Use Search Address field to find a required location.
  4. Click OK button.
    Note: A location cannot be deleted while there are active postings for it.
