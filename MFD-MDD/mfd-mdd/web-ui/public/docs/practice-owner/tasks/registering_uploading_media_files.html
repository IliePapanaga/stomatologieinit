3.2.5 Clone Posting

You can register or upload a media file.

  1. Click Design Studio.
  2. On the Service Templates tab, select the available template, and click Edit Template.
  3. Expand the Media folder, click Media Files, and then Add Media File.
    Remember: If you already have a set-up media store with existing files, then you need to click Register Media File, and select the media store. However, if you are adding the media file from a local URI, click Upload Media File, and select it: both Media Store and Source Path folder will be generated automatically, and the default store will be used.
  4. Specify the following:
    • Media Store - the store you previously set up.
    • Source Path - the path to the media file being added, for example, smb://storage.mydomain.local/path/to/files.
    • Name - the file name; make sure that the file-name and its extension/format are spelled precisely the way they are spelled on its corresponding media store, or the file will not be uploaded.
    • Media Folder - the folder you have set up.
    • Media Type - the type to be applied when a template's service is provisioned.
    • Media Format - the file-name extension.
  5. Click OK.
