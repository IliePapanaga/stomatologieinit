7.2.2 Add/Edit Location

A doctor is able to add more than one location.

  1. Click Settings Tab at the Navigation Panel.
  2. Click Locations at Settings at the Navigation Panel.
  3. Click ADD button.
  4. Observe that ADD LOCATION dialog appears.
  5. Populate the required fields.
    Note: Time Zone is added automatically based on ZIP information.
    Note: At least, one location should be added by a user to publish a job posting.
