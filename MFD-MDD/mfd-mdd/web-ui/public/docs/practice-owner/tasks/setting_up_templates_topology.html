Creating a template's version

You can re-use a preconfigured or already available template to avoid having to set up its parameters, scripts, etc. all over again (note that if a template is in its "Active" state, you cannot create a version of it).

  1. Click Design Studio.
  2. Click either the Service Templates or Application Templates tab, select a template, and then click Create Version on the toolbar.
  3. To specify the template's version, click Browse.
  4. Expand the General folder, click Properties, and, in the Key field, specify a unique short name (P3C automatically validates its uniqueness).
  5. Expand the Service Management folder, click Deployment Profiles, and then Edit Deployment Profile.
  6. Click Default Deployment Group, and specify a new, unique key.
  7. Click OK.
The new version of a template is added to the Templates pane.
