3.2.4 Schedule Job Interview

A doctor is able to schedule a working or personal interview with a candidate.

  1. Select a required Posting.
  2. Click View Applicants.
    Note: If no professional applied to a posting, View Applicants is disabled.
  3. Select a required Professional.
  4. Click SCHEDULE button.
  5. Select Interview Type.
    Note: Working Interview is paid as one day work like as temporary job for the specified posting.
    Note: Personal Interview is not paid by dental practices.

    Remember: At the end of a business day, a doctor should inform Mayday Dental Staffing by email about interview results:

    • A candidate will be hired for a permanent position;

    • An interviewed candidate is rejected;

    • Interview with this or other candidates will be continued.

    Note: Please be informed that if you want to have candidate work any trial days (working interviews) Mayday Dental Staffing should be informed about it.
    Note: Please be informed you are required to inform Mayday Dental Staffing, if you hire a candidate permanently.
    1. Specify four options for Date and Time of Interview.

    Mayday Dental Staffing application provides ability to select four time slots for Date and Time of an interview.

    A doctor should filled out all four options in order to give a chance for a candidate to select an appropriate date and time.

    1. Provide Comments if it is necessary.
  6. Click OK.
  7. Observe the Interview Status.
The interview is schedulled.
Note: Permanent Posting Status for the this posting is Under Review.
