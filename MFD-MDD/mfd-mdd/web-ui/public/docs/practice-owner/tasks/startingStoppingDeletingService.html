4.3 Rejection of a Candidate

An environment can include 1 application and a number of service modules, while a service module can contain lots of separate services. When you click Upgrade on the Environments tab's toolbar, the associated services are upgraded automatically, i.e. you do not have to stop an individual service. However, under some circumstances, you might need to stop, or delete, a discrete service.

Remember: You can stop, and then start, a service only if you had previously added the corresponding operation types (i.e. Shutdown and Startup) when setting up a template, Design Studio > Services > Service Templates > Add Template > Service Management > Operations > Add Operation > General > Properties > Operation Type.
  1. Click Application Operations.
  2. Click the Environments view, then the Services tab.
  3. To delete a service, select it, and click Delete Service.
  4. Note: You can stop only the Active status services.
    To stop or start it, select it, and click Stop Service or Start Service.
