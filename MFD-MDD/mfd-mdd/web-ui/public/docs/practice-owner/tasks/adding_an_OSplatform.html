Adding an OS platform

You can set up the type of an operating system that a provider's VM image will be installed on.

  1. Click Infrastructure Operations.
  2. Click the Deployment view, and then Add OS Platform; enter the following:
    • Family - an OS type, Linux or Windows.
    • Architecture - type in a descriptive label, e.g. 64-bit.
  3. Under the General folder, click VM Images, and then Add VM Image.
  4. Specify the following:
    • Cloud - select one of the clouds that you previously configured in P3C.
    • Site - depending on the selected cloud, choose one of its sites.
    • VM Image - select one of the virtual machine templates available within your proprietary VM environment (e.g. from vSphere, etc): make sure that this VM image does belong to the OS Family you specified earlier because P3C cannot verify that automatically.
    • User name and password - the credentials of VM's default system administrator.
    • Transport Protocol - select either SSH (Linux OS flavors only, unless the SSH server was installed on a Windows image), WinRM (Windows® only), or vSphere (if the cloud you selected in the previous step is VMware vSphere).
  5. Click OK.
