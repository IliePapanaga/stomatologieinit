4.3 Check in Dialog

  1. Navigate to Current Attendance view.
  2. Select a required record with Need check in status.
    Note: Only records with Need check in status are available to make Check in
  3. Click Check In button.
  4. Observe Check in dialog.
  5. Click OK button.
  6. Observe at Current Attendance view that the status is changed.
    Note: Please be advise that it is possible to make Check in one hour before and one hour afer job starts. Otherwise a validation error appeares.
