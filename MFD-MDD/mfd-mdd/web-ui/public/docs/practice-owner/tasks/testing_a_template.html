3.2.3 View Applicants for Permanent position

Before publishing and provisioning a service or application template that can be used in another area, you must test, i.e. provision, it to make sure it is operational.

  1. On the Service Templates or Application Template tab, select the template you need, and click Test Template.
  2. From the Environment Type list, select the environment you had set up in step 1.
  3. From the Space list, select a space the template will be tested against.
  4. In the Deployment Profile field, choose the deployment profile you added when creating a new template or editing the existing one (the template will be tested against the profile you specified).
  5. Click Parameters, and then select a parameter and double-click within the Value column: if necessary, you can type in a different value.
  6. Click Provision.
When a service or application template's testing is completed, "Provisioned" will be displayed in the State column, Lab > Activity History.
