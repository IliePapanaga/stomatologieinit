3.2.6 Cancel Interview

  1. Select a required Posting.
  2. Click View Applicants.
    Note: If no professional applied to a posting, View Applicants is disabled.
  3. Select a required Professional.
  4. Observe the Interview Status is Invitation sent.
  5. Click CANCEL button.
  6. Observe the Cancel Dialog.
  7. Click OK button.
  8. Observe that Interview Status is Cancelled.
  9. Click CLOSE button.
