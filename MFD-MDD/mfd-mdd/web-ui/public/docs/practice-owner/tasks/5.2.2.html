5.2 Hire for Permanent Job

A doctor is able to hire a specific professional based on his personal preferences.

  1. Click Past Hiring History Tab at the Navigation Panel.
  2. Select a professional and click Hire for Permanent Job button.
  3. Observe that ADD/EDIT PERMANENT POSTING dialog appears.
    Note: Please be aware that this functionality for permanent jobs only.
  4. Follow the usual procedure for Permanent Jobs. See details 3.2 Permanent Job Management
    Note: After clicking OK button, a doctor is able to monitor a process of hiring as usual Permanent Job Management but only with one Candidate.
    Note: Please be informed that only selected candidate for the required permanent job is able to see a posting published via Hire for Permanent Job functionality.
