3.2.1 Add/Edit Permanent Posting

A user is able to make a new permanent posting. The Mayday Dental Staffing application supports a number of positions.

  1. Open Postings In Progress/Permanent Job
  2. Click ADD.
  3. Select Category and check Sub-category.
    Note: The Mayday Dental Staffing application supports a number of positions. See Appendix for reference
  4. Select Location.
  5. Select Start Date.
  6. Click SELECT for Work Schedule.
  7. Select required days and specify required start and end time for a day of a week.
  8. Click POST.
  9. Specify a required Language.
    Note: Engilsh is selected by default.
  10. Add Comments if they are required.
  11. Click POST.
A posting is published and the status of the posting is Active.
