7.1 General Settings

You can add various user profiles, depending on their role within the system, and then assign specific roles to them.

If not yet added, you must first add at least one role.

  1. Click System Administration, then the Users tab, and Add User on the toolbar.
  2. Specify the user names, as well as the notification email address.
  3. Click Address, and enter mobile and office phones, street, ZIP code, city, country, and state associated with the user.
  4. Click Roles, then Add Role, and then select the corresponding check-box (you can select more than 1 role).
  5. Click OK.
    Tip: To associate more roles with a user, select the user, and click Assign Roles on the toolbar.
If you need to reset the user's password within the system, select the user, and then click Reset Account: the password-change request will be emailed to the address you entered in User Info when adding the user.
