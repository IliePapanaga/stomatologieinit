3.1.5 Booking Candidate

A doctor is able to hire one of the applied professionals.

  1. Select a required Posting.
  2. Click the View Applicants button
    Note: If no professional applied to a posting, View Applicants is disabled.

    View Applicants View - TBD

  3. Select a professional.
  4. Click BOOK button.
    • As a result, Invitation sent is displayed.

    View Applicants View - TBD

  5. Click OK.
    Note:

    If a candidate accepts the offer and the posting is in Filled status.

    Note:
    • Practice owner can book more than one applicant for the same job posting if professionals applied for the days of the posting that are not overlapped. For example, a posting is created for next days: Monday, Tuesday, Wednesday, Thursday, Friday. One professional applied for Monday and Tuesday, and another one for Wednesday, Thursday and Friday. In this case,a doctor can hire both professionals. However, if the other professional applied for Tuesday, Wednesday, Thursday and Friday, a doctor cannot book both candidates.
    • This intentional limitation should reduce the amount of invitation rejections by professionals.
