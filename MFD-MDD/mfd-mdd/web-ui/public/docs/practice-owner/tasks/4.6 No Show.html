4.6 No Show

If an assigned candidate has not arrived to a dental practice, a Doctor is able to mark this candidate with “No Show” status.

  1. Navigate to Current Attendance view.
  2. Select a required record.
  3. Click NO SHOW button.
  4. Observe NO SHOW dialog.
  5. Click OK button.
    Note: Professionals are informed that after two No Show cases the candidate is not able to see any new postings, however a candidate is able to see the assigned postings.
    Note: MaydayDental Staffing office receives a notification email about “No Show” Status.
