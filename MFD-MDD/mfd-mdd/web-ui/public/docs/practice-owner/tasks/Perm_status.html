3.2.2 Permanent Posting Statuses

There are next available statuses of the permanent postings.
Table 1.
Status Description

Active

Posting is published by the dental office and a start date in future. The candidates may apply to a posting, but a doctor has not booked anyone yet.
Under Review

Some candidates have applied to the posting. A doctor has invited some candidates for an interview (working or personal). Eligible candidates are still able to see the posting. Notifications are sent to invited applicants.
Filled

A doctor has booked a candidate and the candidate accepted the job offer. Notification is sent to a candidate and Mayday Dental Staffing office about it. The posting is not available for the rest of eligible candidates.
Rejected A doctor has booked a specific candidate using functionality “Hire for Permanent Job”. However, the candidate has rejected the offer
