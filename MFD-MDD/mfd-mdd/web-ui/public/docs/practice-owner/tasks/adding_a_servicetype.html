3.1.1 Add/Edit Posting Dialog

You can add to P3C a service type that will serve as an umbrella descriptor for the types of services being rendered, e.g. application or hosting services.

A user is able to make a new posting by following wizard dialogs.

  1. Click on “ADD” button.

    “Posting Name” will be generated automatically after selection of required “Position” and “Location” values, i.e.:

    Note: The Mayday Dental Staffing application supports a number of positions. See Appendix for reference

    After selection of required “Position” and “Location”, a user can select a candidate from two options:

    • “Any Available”. In this case, a professional with appropriate skills will see your posting.
    • “Direct Booking”. In this case, a specifically preferred professional can be hired by clicking “Browse” button and selection the candidate. This posting is NOT available for other candidates
    .
  2. Select “Any Available”.
  3. Select Posting Date & Time Type:

    A doctor may require a professional in different timetable. For this purpose, an application supports three options to select posting date and time:

    • Simple. A user is able to select Start Date and End Date. Time of the beginning and ending the job is the same, if a posting lasts more than one day.
      • Usually, this type of posting is used if a doctor needs a professional for one day or for a number of days if start and end of work is the same for all days.
    • Weekly. A user is able to select Start Date and End Date. A user is able to specify a day of the week based on a week schedule. A user is able to specify start and end time of posting for every day of the week.
      • Usually, weekly posting is used if a doctor needs a professional for some days of the week for one or more weeks and start and end time of work can be different from day to today
    • Complex. A user is able to select start, end date and different time for start and end of job.
      • Usually, complex posting is used if a doctor needs a professional for more than one day and simple and weekly patterns cannot be applied.
  4. Select Language.

    Mayday Dental Staffing allows doctor to specify other than English languages required for a posting.

  5. Provide Comments if it is required for you.

    This section is not mandatory and allows doctor to customize the posting if it is necessary. For example, a doctor can specify recommendations about dress code.

    After population of mandatory fields marked with *, a user should click “Next” to specify Date and Time pattern for posting.

  6. Click Next button.
  7. Based on the previosly selected Posting Date & Time Type, different patterns are available.
    1. Simple Posting type.
    1. Weekly Posting type.
    1. Complex Posting type
