You can add a media folder to a service template. A media folder can contain a range of files and scripts to be activated when the template's service gets provisioned.

  1. Click Design Studio.
  2. On the Service Templates tab, either add a template or select the available one, and click Edit Template.
  3. Expand the Media folder, click Media Folders, and Add Media Folder.
  4. From the Supported Media Types list, select one or more of the following options:
    • Distributive - the media folder type will support distributive files, for example, .rar, .jar, .exe, etc.
    • Script - the folder type will support .ps1 and, for Linux flavors, .sh scripts.
    • Customization Project - if selected, the media folder will also support customizations of applications on a VM, i.e. the customized files; if you select this option, make sure you have also selected the Allow Customizations check box.
  5. Click OK.
