3.1.3 Temporary Posting Statuses

Next temporary posting statuses are available in the application:

Table 1. Temporary Posting Statuses
Status Description
Active Posting is placed by the dental offices and start date in future. If Start date is less than current date – such postings are not listed.

Notifications are sent to all eligible candidates and system admin.

Candidates may apply to a posting, but a Medical Doctor does not book anyone. Candidates are able to apply to another postings as well.
Partially filled Candidates applied to that posting.

Dental Practice has approved some candidate. But not all days are filled out by applicants.

Posting is still available for all eligible candidates on not occupied dates.

Notifications are sent to applicants and Mayday Dental Admin office if any candidate is booked.

Filled

 Dental Practice has booked a candidate and the candidate accepted the job offer.

Notification is sent to candidate and Mayday Dental Admin office about it.

Posting is NOT available for the rest of eligible candidates.

Rejected

 Dental Practice has made a direct booking to hire a specific professional. However, the professional rejected the invitation.
