4.1.1 Temporary Job Attendance Statuses

If you have already provisioned an application or service template, but have to modify it, you can deactivate it, edit/update it from Design Studio, and then re-provision an updated version again .

  1. Click Application Operations.
  2. Click Application Store > Applications, or, if you want to cancel a service template, click Application Store > Modules.
  3. Select a provisioned application or service template, and click either Reject Application or Reject Service.
The corresponding template is removed from the list of active/provisioned templates.
