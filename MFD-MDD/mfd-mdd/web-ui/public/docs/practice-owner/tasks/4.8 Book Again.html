4.8 Book Again

A doctor is able to hire the same candidate.

  1. Navigate to Current Attendance view.
  2. Select a required record.
  3. Click BOOK AGAIN button.
  4. Observe a new POSTING dialog.
  5. Populate the required fields and select the required options. For details see 3.1.1 Add/Edit Posting Dialog.
  6. Click OK button.
