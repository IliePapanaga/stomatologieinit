4.4 Alerts

You can delete the environments with the Active or Ready For Provisioning status, but not with the Processing one. Before deleting an environment, you must first delete all of the templates associated with it.

  1. Click Application Operations.
  2. Click the Environments view, select the environment, and click Edit Environment on the toolbar.
  3. Click Environment > Logical Topology, select the template(s), and click Uninstall, and then OK.
