5.1 Post Reviews

A user is able to post and see the published earlier reviews. A doctor is able to provide review for a professional after each job.

  1. Click Past Hiring History Tab at the Navigation Panel.
  2. Select a professional and click See Review button.
  3. Observe that See Reviews dialog appears.
  4. Select a required record.
  5. Click POST REVIEW button.
    Note: A doctor can provide feedback about professionals in next categories:
    • Professionalism
    • Communication
    • Work Quality
    • Punctuality
    • Appearance

    Total rating of a candidate is available for all doctors and helps to make a decision about hiring an appropriate candidate.

  6. Select stars to specify your feedback.
    Attention:
    • The candidate does not see the review. This information is available only for Mayday Dental Staffing.
    • The review of candidate cannot be deleted by a doctor, so please be careful about providing feedback
    Tip: If you want to hire a professional for a long term, select I would hire this PRO permanently.
  7. Provide Comments, if you want to give some details.
  8. Click OK.
