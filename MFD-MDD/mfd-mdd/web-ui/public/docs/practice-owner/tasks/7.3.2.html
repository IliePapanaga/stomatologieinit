7.3.2 Add/Edit Payment Method Dialog

  1. Navigate to Payment Methods Tab at the Settings at Navigation Panel.
  2. Click ADD to start adding a new payment method.
  3. Observe Payment Method Dialog.
    Note: Please be informed that the selected payment method covers all your offices.
    Note: A user should select one of the types of payment method:
    • Credit Card
    • Bank Account
      Note: Please be informed that one of the payment method should be selected as preferred.
  4. Select Credit Card
    or Select Bank Account
  5. Populate the required fields.
  6. Click OK button.
