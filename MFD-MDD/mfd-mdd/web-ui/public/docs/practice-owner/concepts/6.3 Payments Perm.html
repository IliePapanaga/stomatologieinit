6.3 Payments for Permanent Positions

The payment amount for the permanent positions is calculated by next formula:

A = Cfs*Wh*Wpy*Nwd*Ph,

where

A = Payment Amount for a permanent job;

Cfs = Compensation fee for specialty;

Wh = Working hours = 8 hours;

Wpy = Weeks per year = 52;

Nwd = Number of working days per week. Values from 1 to 7;

Ph = Percentage for hiring (15%).

For example, let’s calculate payment amount (A) for RDH (Cfs =$ 60), if a professional is hired for 5 days per week (Nwd = 5):

A = 60*8*52*5*0.15 = $ 18,720

So, the payment amount for the permanent position (RDH) is $ 18,720.
