6.1 Payments View

A user is able to track the payments based on Posting name, Type of posting (temporary or permanent), Professional Name (Last and First Names), Date of the payment, used Payment Method (Credit card –CC, or Bank account), an amount of money, Status of payment.

Note: If a status of a payment is “Failed”, please check:
  • your available 7.3.1 Payment Methods View
  • verify expiration date of a credit card;
  • available amount of money.

    If you have not found a reason for a failed payment, please contact Mayday Dental Staffing.
